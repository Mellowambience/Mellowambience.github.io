Security architecture

Most AI extensions see everything.
AetherTab sees only what it needs.

In 2025, 300+ malicious AI extensions were caught harvesting credentials and browsing history. This was built as the direct response.

API key storagePlaintext in chrome.storageAES-GCM 256-bit encrypted. PBKDF2, 100k iterations, unique salt per install.
Host permissions<all_urls> broad accessExplicit allowlist — 5 domains only. No wildcard.
AI output renderingRaw innerHTML (XSS vector)textContent only. No innerHTML from AI responses. Ever.
Content Security PolicyMissing or weakStrict script-src 'self'. No eval. No inline scripts.
AnalyticsGoogle Analytics, Mixpanel, etc.None. Zero. Not a tracker in sight.
